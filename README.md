# steady point

학교 전용 공부 타이머 및 보상 플랫폼의 동작형 프론트엔드 프로토타입입니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`으로 접속합니다.

## 주요 기능

- 학생/관리자 모드 전환
- 학교 코드 인증 기반 학생 가입
- 개인정보 및 기기 접근 권한 필수 동의
- 30분 단위, 30분~6시간 일반 타이머
- Page Visibility API 기반 5초 화면 이탈 무효 처리
- 타이머 정상 종료 시 포인트 적립
- 포인트 상품 구매 및 지갑 보관
- 20초마다 갱신되는 일회용 결제 코드
- 월간 공부 그래프 및 성장률 보상
- 관리자 상품 CRUD 및 이미지 파일 업로드/미리보기
- 학교 이름/코드 생성 및 삭제
- 학생별 포인트 직접 수정
- 보상 배율과 성장률 기준 설정
- Mock 스캐너를 통한 기프티콘 사용 완료 처리
- localStorage 기반 상태 유지

## 테스트 팁

기본 가입 상태는 완료로 되어 있습니다. 회원가입 화면을 다시 보려면 브라우저 개발자 도구에서 `steady-point-state` localStorage를 삭제하고 코드의 초기 `signedUp` 값을 `false`로 변경하세요.
